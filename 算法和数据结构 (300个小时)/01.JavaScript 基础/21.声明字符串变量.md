## 声明字符串变量

> 之前我们写过这样的代码：
>
> ```js
> var myName = "your name";
> ```
>
> `"your name"`被称作字符串变量。字符串是用单引号或双引号包裹起来的一连串的零个或多个字符。

---

> 创建两个新的`字符串变量`：`myFirstName`和`myLastName`，并用你的姓和名分别为它们赋值。

```js
// 示例
var firstName = "Alan";
var lastName = "Turing";

// 请把你的代码写在这条注释以下
var myFirstName = "z";
var myLastName = "cj";
```
