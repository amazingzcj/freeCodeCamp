## while 循环

> 你可以使用循环多次执行相同的代码。
>
> 我们将学习的第一种类型的循环称为 "`while`" 循环，因为它规定，当 "while" 条件为真，循环才会执行，反之不执行。
>
> ```js
> var ourArray = [];
> var i = 0;
> while(i < 5) {
>   ourArray.push(i);
>   i++;
> }
> ```
>
> 让我们通过 while 循环将值添加到数组中。

----

> 通过一个`while`循环，把从 0 到 4 的值添加到`myArray`中。

```js
// 初始化变量
var myArray = [];

// 请把你的代码写在这条注释以下
var i=0;
while(i<5){
    myArray.push(i);
    i++;
}

```
